Payreq Reckon Accounts Hosted Usage

The only information accessed by Payreq from your Reckon Hosted account is:

  1. Your general ledger account codes to allow you select an account code for your invoice. Only the ID of the account code selected is stored in Payreq.
  2. Your suppliers to allow you to select a supplier for your invoice. Only the ID of the account code selected is stored in Payreq.
  3. Your tax codes to allow you to select a sale tax code for your invoice. Only the ID of the tax code selected is stored in Payreq.

No other information from Reckon is accessed by Payreq, registered billers or any other third party.

Payreq will use your Reckon connection to push biller invoices into your Reckon accounts payable.
